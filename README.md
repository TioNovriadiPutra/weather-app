## Getting Started

### Prerequisites
- Node >= 16.14.0
- Android Studio

### Installation
1. Clone the repository :
   
   ```bash
   git clone https://github.com/TioNovriadiPutra/weather-app.git
2. Navigate to weather-app folder :

   ```bash
   cd weather-app
3. Install the required dependencies:

   ```bash
   npm install

### Run Program
1. Run the Android Emulator
2. Navigate to weather-app folder :

   ```bash
   cd weather-app
3. Run :

   ```bash
   npm run android
